Game Size: Defualt set to 4 , can be configured in GAME_CONF.js
 

Controls Conf: Defualt set to "numPadControl", can be configured in GAME_CONF.js		 
    Available Settings: 
    i. “arrowsControl” for Arrow movement.
    ii. “numPadControl” for number movement. 		
	
By Default game is configured to 4*4, configuration can be changed to 8*8 or 16*16 inside GAME_CONF.js

Steps to Start :

1.	Launch “StartPlay.html” in browser (Best viewed in Chrome).
2.	Enjoy the playing using the controls.
